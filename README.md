## 🛠 Getting Started

If you are contributing to this project:

1. **Clone the repository**
2. **Initialize your project locally** inside the repository folder  
   (for example: React, Next.js, Vite, etc.)
3. Create a **new branch** for your work (based off our Git workflow guide)
4. Push your changes and open a **Pull Request (PR)**

Example:

```bash
git clone <repo-url>
cd team_portfolio

# initialize project
npm create vite@latest
# or
npx create-next-app .

# create branch
git checkout -b setup/project-initialization

git add .
git commit -m "Initial project setup"
git push origin setup/project-initialization
```

Then open a **Pull Request** to merge into `main`.

## 📌 Important Rule

⚠️ **Do NOT push directly to `main`.**

All major changes, including project initialization, must go through a **Pull Request** so the team can review them.
